## Deployment

This is a static website that can be deployed using any of these options:

### Cloudflare Pages (Recommended - Unlimited Bandwidth)
1. Sign up at [pages.cloudflare.com](https://pages.cloudflare.com)
2. Click "Create a project" â†’ "Connect to Git"
3. Authorize Cloudflare to access your GitHub/GitLab repo
4. Select your repository and configure:
   - Build command: *(leave empty for static sites)*
   - Build output directory: `/` or `.`
5. Click "Save and Deploy"
6. Your site will be live at `https://project-name.pages.dev`

### GitHub Pages
1. Push this repository to GitHub
2. Go to repository Settings â†’ Pages
3. Select your branch (usually `main`) and save
4. Your site will be live at `https://username.github.io/repo-name`

### Netlify
1. Visit [netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop the project folder
3. Your site is live instantly

### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project directory
3. Follow the prompts to deploy
